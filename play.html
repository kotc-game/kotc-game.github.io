Starter Set

Since Keeper of the Cards is inherently a DIY game, you will need a toolbox to get started. The Keeper of the Cards starter set exists to facilitate that. It contains starter decks you can print, as well as everything you need to get started making your own cards. You can find the latest starter set here:
